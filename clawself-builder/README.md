# ClawSelf Builder（灵魂提取器）技能说明

## 是什么

ClawSelf 第一层：**灵魂提取**。通过 100 问灵魂提取协议（六维度）逐问访谈用户，记录原始回答到 `raw_interview_data.log`，并生成结构化的 `persona.md`，作为数字分身的人设底座。

- **六维度**：投资哲学与生存本能、语言风格与社交黑话、逻辑推演与认知偏见、情绪触发与压力反应、币安生态与社区认知、个人背景与未来愿景。
- **persona 六模块**：【人格底色】【交易逻辑】【社交黑话库】【情绪触发机制】【币安生态与社区认知】【个人背景与愿景】+ 访谈原文（Q1–Q100）。

## 何时用

- 需要「灵魂提取」「分身人设」「数字人格画像」「生成 persona.md」时。
- 首次搭建 ClawSelf 分身、或需要重新生成/更新人设时。

## 技能文件

- **clawself-builder.md**：完整流程（初始化 → Q1–Q100 逐问 → 日志格式 → 生成 persona 规则 → 交付检查）、100 题清单与 persona 输出模板。
- **README.md**：本说明。

## 产出

- **raw_interview_data.log**：Q1–Q100 的问答原文（可 SKIPPED/SANITIZED），可追溯。
- **persona.md**：结构化人格画像，供感知层高优先级匹配、内容层人设注入、执行层与总控使用。
